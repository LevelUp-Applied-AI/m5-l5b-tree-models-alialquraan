# Lab 5B — Trees & Ensembles

Module 5 Week B lab for AI.SPIRE Applied AI & ML Systems.

## Setup

```bash
pip install -r requirements.txt
```

## Tasks

Complete the functions in `lab_trees.py` to build and evaluate decision tree and random forest models with class imbalance handling. See the lab guide for detailed instructions.

## Submission

Your PR description must include:
1. Classification report for default vs. balanced random forest
2. Top 5 features by importance
3. PR-AUC values for both models
4. Brief comparison to Week A logistic regression (~3 sentences)
5. Paste your PR URL into TalentLMS → Module 5 Week B → Lab 5B to submit this assignment
